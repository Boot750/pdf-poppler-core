# pdf-poppler-binaries-darwin

macOS binaries for [pdf-poppler-core](https://npmjs.com/package/pdf-poppler-core).

This package contains pre-compiled Poppler utilities for macOS (x64).

## Installation

```bash
npm install pdf-poppler-binaries-darwin
```

**Note:** You also need to install `pdf-poppler-core` to use these binaries.

```bash
npm install pdf-poppler-core pdf-poppler-binaries-darwin
```

## What's Included

- Poppler utilities compiled for macOS x64
- All required dynamic libraries

## System Requirements

- macOS x64
- No additional system dependencies required
